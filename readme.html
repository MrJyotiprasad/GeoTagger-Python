📍 GeoTagger-Python

v1.0 A Python tool to auto-geotag your images and add a stylish map + location overlay at the bottom corner. Great for events, travel, or branding your media.

🛠️ Features

  • ✅ Adds GPS EXIF metadata to all JPEG/PNG images in a folder
  • 🗺️ Pastes a mini map with event/location/weather details
  • 🎯 Perfect for event-based or testing photo archives
  • ⚡ Fast, offline, no API keys needed

🚀 How to Use

  1. Place your JPEG/PNG files in the same folder as the script
  2. Run the script using Python 3:
pip install -r requirements.txt
python geostamp.py

📦 Requirements

Pillow==10.3.0
piexif==1.1.3
staticmap==0.5.4
